# BillingResourcesStore

Let users buy resource packages or individual resources using credits. Integrates with **Billing Core** for payments and **Billing Resources** to add purchased limits to the user's pool.


## Features

- **User-facing**
  - **Billing → Resource Store** (`/resources/store`)
  - Browse resource packages; purchase with credits
  - Buy individual resources à la carte (memory, CPU, disk, etc.)
  - View purchase history; live credit balance from Billing Core

- **Admin**
  - **Fremium Resources → Package Store** — create/edit/delete resource packages (bundled limits + price in credits)
  - **Store Settings** — global discount, minimum purchase for discount, bulk discount tiers
  - **Individual Store** — per-resource unit pricing and purchase options
